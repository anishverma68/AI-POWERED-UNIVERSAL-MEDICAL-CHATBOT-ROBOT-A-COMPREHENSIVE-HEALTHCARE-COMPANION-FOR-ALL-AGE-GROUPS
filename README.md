AI-Powered Universal Medical Chatbot Robot
🏥 A Comprehensive Healthcare Companion for All Age Groups
🌟 Overview
This project outlines the design and development of an innovative Medical Chatbot Robot. It functions as a comprehensive, AI-driven healthcare companion system.

While many existing health solutions cater to specific demographics, this system is designed to be universal, meeting the healthcare requirements and accessibility needs of all age groups—from children to the elderly.

🚀 Key Features
AI Chatbot Interface: Real-time medical assistance and query handling using chatbot.js.

Universal Design: A user-friendly interface (homepage.html) accessible to both tech-savvy users and the elderly.

Multilingual Support: Integrated translation features via custom_translate.js to assist users in their preferred language.

Health Reminders: Automated system for medication and check-up alerts using reminders.js.

Interactive UI: Engaging experience with dedicated animation and icon assets to make healthcare interaction less intimidating.

📂 Project Structure
The repository is organized as follows:

Bash
├── Interface/           # UI design components and layouts
├── animation/           # Animation assets for the chatbot interaction
├── icon/                # Visual icons used throughout the application
├── chatbot.js           # Core AI logic and chatbot processing
├── custom_translate.js  # Language translation and localization logic
├── homepage.html        # The main landing page and user interface
├── reminders.js         # Logic for managing medical and health alerts
└── style.css            # Global styling and responsive design
🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript

Interactivity: Custom JS Animations

Logic: Vanilla JavaScript (AI integration ready)
