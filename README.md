# AI-Powered Universal Medical Chatbot Robot
### 🏥 A Comprehensive Healthcare Companion for All Age Groups

## 🌟 Overview
This project outlines the design and development of an innovative **Medical Chatbot Robot**, which functions as a comprehensive, AI-driven universal healthcare companion system. 

In contrast to existing solutions that cater to specific demographics, our proposed system meets healthcare requirements for all age groups—from children to the elderly—by providing an accessible and intuitive interface.

## 🚀 Key Features
- **AI Chatbot Logic:** Core interaction handling via `chatbot.js`.
- **Universal Accessibility:** A responsive `homepage.html` designed for ease of use across all ages.
- **Multilingual Support:** Integrated translation capabilities using `custom_translate.js`.
- **Smart Reminders:** Automated health and medication alerts managed by `reminders.js`.
- **Visual Engagement:** Custom animations and icons to provide a friendly "Robot" persona.
